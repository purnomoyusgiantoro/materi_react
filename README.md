Materi ini dirancang khusus untuk **siswa SMA** yang ingin belajar membuat website modern menggunakan **React** (Frontend) dan **Express** (Backend). Tidak perlu pengalaman sebelumnya — kita mulai dari nol!

## 🎯 Tujuan Pembelajaran

Setelah mengikuti 3 jam pelajaran ini, siswa mampu:

1. ✅ Memahami konsep dasar **Frontend** dan **Backend**
2. ✅ Membuat **server API** sederhana menggunakan Express.js
3. ✅ Membuat **tampilan web interaktif** menggunakan React
4. ✅ **Menghubungkan** React (Frontend) dengan Express (Backend)
5. ✅ Membuat **aplikasi Todo List Full-Stack** yang berfungsi penuh

## 🛠️ Persiapan Sebelum Mulai

### Software yang Perlu Diinstall:

1. **Node.js** (versi LTS) — [Download di sini](https://nodejs.org/)
2. **Visual Studio Code** — [Download di sini](https://code.visualstudio.com/)
3. **Browser** (Chrome/Edge/Firefox)

### Cara Cek Apakah Node.js Sudah Terinstall:

Buka **Terminal/Command Prompt**, ketik:

```bash
node --version
npm --version
```

Jika muncul nomor versi (misal `v20.x.x`), berarti sudah berhasil! 🎉

## 💡 Tips Belajar

> **🔑 Jangan takut salah!** Error itu hal biasa dalam programming. Yang penting baca pesan error-nya dan coba perbaiki.

> **📝 Ketik sendiri, jangan copy-paste!** Mengetik kode sendiri membantu kamu mengingat lebih baik.

> **🤝 Tanya teman atau guru!** Kalau stuck, jangan malu bertanya.
