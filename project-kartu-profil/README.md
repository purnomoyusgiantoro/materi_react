## ⚡ Quick Start (Mulai Sekarang!)

### Langkah 1: Copy Template
```bash
# Copy folder template ke folder baru
cp -r template-frontend kartu-profil-frontend
cp -r template-backend kartu-profil-backend
```

### Langkah 2: Install & Jalankan Backend
```bash
cd kartu-profil-backend
npm install
node index.js
```

### Langkah 3: Install & Jalankan Frontend (buka terminal baru!)
```bash
cd kartu-profil-frontend
npm install
npm run dev
```

### Langkah 4: BUKA BROWSER! 🎉
Buka `http://localhost:5173` dan... **BOOM!** Kartu profil keren langsung muncul! 🤯

> 📖 **Sekarang buka `panduan-misi.md` untuk mulai petualangan!**
